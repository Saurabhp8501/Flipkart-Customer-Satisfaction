# Flipkart Customer Satisfaction Analysis (EDA & Machine Learning)

## Project Overview
This project analyzes Flipkart customer support data using Python. It includes exploratory data analysis, feature engineering, visualization, statistical analysis, and machine learning models to understand the factors affecting Customer Satisfaction (CSAT).

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- imbalanced-learn

## Project Highlights
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Feature Engineering
- Logistic Regression Classification
- SMOTE for class balancing
- Data Visualizations
